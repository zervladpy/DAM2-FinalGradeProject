## Description

La idea detrás de este proyecto es hacer la vida más fácil en la gestión de centros ecuestres.
Este servicio proporcionará las siguientes características básicas:

- Gestión de usuarios
- Gestión de caballos
- Gestión de horarios
- Gestión de clases
- Gestión de eventos
- Gestión de organizaciones
